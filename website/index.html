What it does Mit tud az eszköz

scan2epub converts scanned or photographed PDF books into clean, readable EPUBs. It uses Azure OCR to extract text and AI to intelligently clean up OCR artifacts — optimized for Hungarian. A scan2epub a szkennelt vagy fotózott PDF könyveket tiszta, jól olvasható EPUB formátumba konvertálja. Azure OCR-t használ a szöveg kinyerésére, és AI-t az OCR hibák intelligens javítására — magyar nyelvre optimalizálva.

PDF → EPUB

Accepts local PDF paths and public URLs. Local files are uploaded to Azure Blob to generate a temporary SAS URL automatically.

Helyi PDF fájlokat és nyilvános URL‑eket is elfogad. A helyi fájlokat az eszköz automatikusan Azure Blob‑ba tölti fel ideiglenes SAS URL létrehozásához.

OCR + Cleanup

Azure OCR for text extraction; AI removes hyphenation, stray line breaks, and artifacts.

Azure OCR a szöveghez; az AI eltávolítja a kötőjeleket, felesleges sortöréseket és OCR hibákat.

Flexible Modes

Full pipeline, OCR-only, or cleanup-only — choose what you need.

Teljes folyamat, csak OCR vagy csak tisztítás — igény szerint választható.

Key features Fő funkciók

  • Hungarian‑first cleanup: handles hyphenation rules and special characters. kezeli a magyar elválasztási szabályokat és a speciális karaktereket.
  • Clean EPUB formatting: preserves book structure for e‑readers. megőrzi a könyv szerkezetét e‑olvasókhoz.
  • Progress & errors: clear feedback during long operations. áttekinthető visszajelzés hosszú feldolgozás közben.
  • Flexible CLI: full, OCR‑only, and cleanup‑only modes. teljes, csak OCR és csak tisztítás módok.

Example

Példa

# Convert (default) — local file or URL
scan2epub "C:\path\to\book.pdf" "C:\path\to\output.epub" --language hu
# or explicit:
scan2epub convert "C:\path\to\book.pdf" "C:\path\to\output.epub"

# OCR only → raw EPUB
scan2epub ocr "C:\path\to\book.pdf" "C:\path\to\raw.epub" --language hu

# Cleanup only → cleaned EPUB
scan2epub clean "C:\path\to\raw.epub" "C:\path\to\final.epub" --save-interim

FAQ

Why do I need a public PDF URL? Miért kell nyilvános PDF URL?

Azure Content Understanding requires a URL for OCR. Local files are supported: the tool uploads your PDF to Azure Blob and uses a temporary SAS URL automatically.

Az Azure Content Understanding az OCR‑hez URL‑t igényel. A helyi fájlok támogatottak: az eszköz feltölti a PDF‑et Azure Blob‑ba, és automatikusan ideiglenes SAS URL‑t használ.

How does the Hungarian cleanup work? Hogyan működik a magyar nyelvi tisztítás?

AI removes OCR artifacts like hyphenation at line breaks, stray line breaks, and page numbers, preserving structure and characters.

Az AI eltávolítja az OCR hibákat (sorközi elválasztás, felesleges sortörések, oldalszámok), megőrizve a szerkezetet és karaktereket.

What about costs? Mi a helyzet a költségekkel?

The tool relies on Azure services which may incur API costs per book; roadmap includes local AI to remove this dependency.

A megoldás Azure szolgáltatásokat használ, ami könyvenként API‑költséget jelenthet; a terv a helyi AI bevezetése a függőség csökkentésére.

Can it handle large books? Kezel nagy könyveket?

Yes, with memory‑aware processing and progress feedback; extremely large inputs may still be constrained.

Igen, memória‑tudatos feldolgozással és státuszjelzéssel; rendkívül nagy fájloknál lehetnek korlátok.
